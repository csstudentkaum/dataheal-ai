# DataHeal AI — تصحيح البيانات الدلالي الذكي

**DataHeal AI** هو نموذج أولي لنظام يعتمد على الذكاء الاصطناعي لاكتشاف التناقضات المنطقية والدلالية في بيانات الاستبيانات قبل استخدامها في التحليل.

🔗 **عرض المشروع:**
https://dataheal-ai.onrender.com

---

## المميزات

* رفع بيانات الاستبيان بصيغة **CSV** أو **JSON**
* تحليل السجلات لاكتشاف التناقضات المنطقية
* لوحة تحكم لعرض النتائج ودرجات الثقة
* تحميل نتائج التحليل بصيغة **CSV**
* توليد بيانات تجريبية لاختبار النظام
* واجهة عربية تدعم **RTL**

---

## كيف يعمل النظام

1. **رفع البيانات**
   يقوم المستخدم برفع ملف الاستبيان بصيغة CSV أو JSON.

2. **تحليل السجلات**
   يتم تحليل كل سجل لاكتشاف التناقضات المنطقية بين الحقول.

3. **عرض النتائج**
   تظهر النتائج في لوحة التحكم مع درجة الثقة والتفسيرات.

## الخوارزميات المستخدمة

### 1. التحقق باستخدام نموذج لغوي كبير (LLM)
- يتم إرسال بيانات كل سجل إلى **OpenAI GPT-3.5 Turbo** عبر API
- يُستخدم أسلوب **Few-Shot Prompting** بتقديم أمثلة توضيحية للنموذج لتوجيه التحليل
- يقوم النموذج بتحليل العلاقات المنطقية بين الحقول وإرجاع **درجة ثقة** وقائمة بالتناقضات المكتشفة

### 2. التحقق بالقواعد المنطقية (Rule-Based Validation)
نظام بديل يعمل بدون API، يطبق القواعد التالية:

| القاعدة | الوصف |
|---------|-------|
| العمر مقابل الخبرة | سنوات الخبرة لا تتجاوز `العمر - 16` |
| المسمى الوظيفي مقابل التعليم | المناصب القيادية تتطلب مؤهلات أعلى من الثانوية |
| الجنس مقابل الحمل | حالة الحمل غير منطقية مع الجنس الذكر |
| العمر مقابل الحالة الاجتماعية | العمر أقل من 14 لا يتوافق مع الزواج أو الطلاق |
| العمر مقابل المنصب | العمر أقل من 25 لا يتوافق مع المناصب القيادية العليا |
| الدخل مقابل الخبرة | الدخل المرتفع (>50,000) غير منطقي مع خبرة أقل من سنتين |

### 3. تحديد حالة السجل
يتم تصنيف كل سجل بناءً على درجة الثقة وعدد المشاكل:

- ✅ **بيانات سليمة** — درجة الثقة ≥ 0.85 وبدون مشاكل
- ⚠️ **تحتاج مراجعة** — درجة الثقة بين 0.50 و 0.85
- ❌ **خطأ منطقي** — درجة الثقة أقل من 0.50

## تشغيل المشروع محليًا

### إنشاء البيئة وتثبيت المتطلبات

```bash
cd dataheal_ai
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### إعداد قاعدة البيانات

```bash
python manage.py migrate
```

### تشغيل الخادم

```bash
python manage.py runserver
```

ثم افتح المتصفح:

http://127.0.0.1:8000
