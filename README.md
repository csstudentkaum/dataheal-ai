# DataHeal AI — تصحيح البيانات الدلالي الذكي

**DataHeal AI** هو نموذج أولي لنظام يعتمد على الذكاء الاصطناعي يساعد على اكتشاف التناقضات المنطقية والدلالية داخل بيانات الاستبيانات قبل استخدامها في التحليل.
بدل أن تكتشف الأخطاء بعد جمع البيانات، يقوم النظام بتحليل الإجابات فورًا واكتشاف أي تناقضات غير منطقية مثل عدم توافق العمر مع سنوات الخبرة أو المؤهل العلمي مع المسمى الوظيفي.

🔗 **لتجربة المشروع مباشرة:**
https://dataheal-ai.onrender.com

---

## ماذا يقدم النظام؟

### 📤 رفع بيانات الاستبيان

يمكنك رفع ملفات **CSV** أو **JSON** تحتوي على سجلات الاستبيان ليقوم النظام بتحليلها.

### 🤖 تحليل ذكي للبيانات

يقوم النظام بتحليل كل سجل واكتشاف التناقضات باستخدام:

* نموذج لغوي كبير (LLM) عند توفر API
* أو نظام تحقق مبني على قواعد ذكية

### 📊 لوحة تحكم تفاعلية

عرض النتائج بشكل بصري واضح مع:

* درجة الثقة لكل سجل
* المؤشرات اللونية للحالة
* تفاصيل الأخطاء المكتشفة

### ⚡ تحقق لحظي عبر API

يمكن ربط النظام مع منصات الاستبيانات للتحقق من صحة البيانات **بشكل فوري أثناء الإدخال**.

### 📄 تقارير قابلة للتحميل

يمكن تحميل نتائج التحليل بصيغة **CSV** لمراجعتها أو استخدامها لاحقًا.

### 🌍 واجهة عربية كاملة

تم تصميم الواجهة لتدعم:

* اللغة العربية
* اتجاه RTL
* تجربة استخدام واضحة وبسيطة

# تشغيل المشروع محليًا

## 1. إنشاء بيئة افتراضية

```bash
cd dataheal_ai
python3 -m venv venv
source venv/bin/activate
```

---

## 2. تثبيت المتطلبات

```bash
pip install -r requirements.txt
```

---

## 3. إعداد قاعدة البيانات

```bash
python manage.py migrate
```

---

## 4. (اختياري) إضافة مفتاح OpenAI

إذا أردت استخدام تحليل يعتمد على نموذج لغوي:

افتح ملف `.env` وأضف:

```
OPENAI_API_KEY=your-api-key
```

إذا لم يتم إضافة المفتاح، سيعمل النظام باستخدام **محرك تحقق مبني على قواعد ذكية**.

---

## 5. تشغيل الخادم

```bash
python manage.py runserver
```

---

## 6. فتح المشروع

افتح المتصفح على:

http://127.0.0.1:8000

---

# استخدام واجهة API

يمكنك إرسال سجل للتحقق منه مباشرة عبر API.

مثال باستخدام curl:

```bash
curl -X POST http://127.0.0.1:8000/validate-record \
  -H "Content-Type: application/json" \
  -d '{"age": 22, "job_title": "Senior Engineer", "years_experience": 12}'
```

سيقوم النظام بتحليل البيانات وإرجاع نتيجة التحقق ودرجة الثقة.
